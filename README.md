# ML-based-optimization-of-Frequency-Selective-Surface-operating-in-X-band
ML-based Band Pass Filter (7GHz) using polygonal FSS on Rogers RO3203. Achieves 0dB transmission and miniaturization 10 x 10 mm to prevent grating lobes. Features a simplified ML model with only 0.5% error margin for rapid, high-power RADAR design.
